## 📋 תיאור הפרויקט / Project Description

פרויקט זה כולל:

1. **מחשבון התאמה אינטראקטיבי** (`calculator.html`) - כלי HTML/JavaScript טהור המחשב התאמה בין העדפות המשתמש לבין 6 קטגוריות התמחות שונות
2. **דוח ניתוח סטטיסטי מקיף** (`reports/analysis_report.html`) - ניתוח מעמיק של סקר 367 מתמחים ברפואה בישראל
3. **דף נחיתה** (`index.html`) - נקודת כניסה מרכזית לכל הכלים

### 🎯 מטרת הפרויקט

לסייע לסטודנטים לרפואה ולמתמחים:
- להבין אילו גורמים משפיעים על בחירת התמחות
- להשוות בין התמחויות שונות על בסיס העדפות אישיות
- לקבל החלטות מושכלות על בסיס נתונים אמיתיים

## 🧮 איך עובד המחשבון? / How the Calculator Works

המחשבון משתמש באלגוריתם המשקלל את ההעדפות שלך מול נתוני הסקר:

### שלב 1: משקלות
כל קריטריון מקבל משקל יחסי לפי החשיבות שנתת לו (1-5):
```
משקל = (ציון שנתת / 5) / סכום כל הציונים
```

### שלב 2: חישוב התאמה
- **קריטריונים ישירים** (איזון, שכר, מחקר, משמעות): `התאמה = ציון ההתמחות / 5`
- **קריטריוני קרבה** (עבודה ידנית, קשר מטופלים, תורנויות): `התאמה = 1 - |העדפה - ציון| / 4`
- **סטרס**: לוגיקה הפוכה (אם מוכן ללחץ → התמחויות עם פחות רוגע)

### שלב 3: ציון סופי
```
ציון = Σ (משקל × התאמה × 100)
```

### בונוסים
- **בונוס הורות**: אם מתכנן הורות → התמחויות עם אחוז גבוה של הורים מקבלות בונוס

## 📊 נתוני הסקר / Survey Data

הפרויקט מבוסס על סקר מקיף שנערך בקרב **367 מתמחים ברפואה בישראל**:

- **תקופת איסוף**: ספטמבר 2024 - נובמבר 2024
- **שיעור היענות**: 72%
- **קטגוריות התמחות**: 6 (כירורגית, פנימית, משפחה, ילדים, פסיכיאטריה, הרדמה)
- **אחוז ממליצים**: 74%

### קטגוריות התמחות
1. **כירורגית** - עבודה ידנית אינטנסיבית, תורנויות רבות
2. **פנימית** - חשיבה אנליטית, קשר מתמשך עם מטופלים
3. **משפחה** - איזון בית-עבודה מצוין, עבודה בקהילה
4. **ילדים** - עבודה עם אוכלוסייה ייחודית, תחושת משמעות גבוהה
5. **פסיכיאטריה** - קשר עמוק עם מטופלים, עבודה פרטית
6. **הרדמה** - עבודת צוות, שכר גבוה יחסית
